# Smoke Checklist

## Запуск и авторизация
- [ ] Приложение запускается (`python main.py`).
- [ ] Валидный логин открывает главное окно.
- [ ] Невалидный логин показывает сообщение.

## Навигация
- [ ] Открываются разделы: Главная, Монитор ТА, Детальные отчеты, Учет ТМЦ.
- [ ] Администрирование раскрывается/скрывается.
- [ ] Профиль сотрудника показывается через кнопку.

## Dashboard
- [ ] Рендерятся 5 блоков.
- [ ] Переключение amount/quantity обновляет график.
- [ ] Плитки перетаскиваются.

## Администрирование ТА
- [ ] Таблица и плиточный режим переключаются.
- [ ] Фильтр, лимит, пагинация работают.
- [ ] Удаление требует подтверждения.
- [ ] Отвязка модема приводит к `-1`.
- [ ] Экспорт в CSV/PDF/HTML создает файлы.

## Монитор ТА
- [ ] Фильтры применяются по кнопке.
- [ ] `Итого автоматов` и `Денег в автоматах` пересчитываются.
- [ ] Пустой результат показывает уведомление.

## SQL ограничения
- [ ] Дубли serial/inventory блокируются.
- [ ] Невалидные даты и диапазоны блокируются.
- [ ] next_verification_date вычисляется триггером.
