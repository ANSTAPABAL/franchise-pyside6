# Руководство пользователя

## Вход
- Запустите `python main.py`.
- Укажите email и пароль.
- При успешном входе открывается главная страница.

## Главная
- Отображаются 5 плиток: эффективность, состояние сети, сводка, динамика, новости.
- Переключайте метрику динамики: `amount`/`quantity`.
- Плитки можно перетаскивать.

## Администрирование -> Торговые автоматы
- Таблица включена по умолчанию; можно переключиться в плитки.
- Доступны фильтр, лимит строк, пагинация.
- Действия: редактировать, удалить, отвязать модем.
- Экспорт в CSV/PDF/HTML.

## Монитор ТА
- Настройте фильтры и нажмите `Применить`.
- Проверьте поля `Итого автоматов` и `Денег в автоматах`.
- При пустой выборке появляется информационное сообщение.
